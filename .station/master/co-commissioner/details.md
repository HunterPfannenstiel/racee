# Co-Commissioner — Details

## What it is
A delegation mechanism that allows the commissioner to share league management responsibilities with other league members.

## Role Assignment
- Only the commissioner can promote an existing league member to co-commissioner
- Only the commissioner can demote a co-commissioner
- No limit on the number of co-commissioners in a league

## Permissions

### Co-commissioner CAN do
- Everything the commissioner can do, except what's listed below

### Commissioner-only actions
- Promote/demote co-commissioners
- Delete the league
- Transfer league ownership

## Phase 0 Scope
- No special UI treatment for co-commissioners (no badges, labels, or notifications)
- No audit trail for who performed an action
- Co-commissioners can only be promoted from existing league members (no inviting someone directly as a co-commissioner)
- Co-commissioners simply gain access to the same management actions as the commissioner
