# Teammate Lineup Setting

My client has sent in a feature request with the following details: We need a way for players to scroll through thier teammates on the predict page and then be able to set thier teamates line up. something like "< "PLAYER" >" would be fine!

The core idea is that sometimes teammates aren't able to submit their line up due to not having cell services, etc... The goal for this feature is to allow for teammates to set each other's lineups and submit them on behalf of the teammate.

It shouldn't matter whether the line up was submitted by the actual player or by their teammate, it should be treated the exact same.
